# 🛠 Server Technology Stack

This repository provides an overview of the technology stack used for our server-side implementation. Below is a breakdown of the frameworks, deployment methods, databases, and tools used.

## 🧱 Framework
- **SpringBoot**: Framework for building Java-based web applications and microservices.
- **Spring Data JPA**: Abstraction layer for JPA implementations to interact with relational databases.

## 🚀 Deployment
- **Amazon EC2**: Cloud platform service to deploy and manage our application servers.

## 🗄 Database
- **MySQL**: Relational database management system for storing structured data.
- **Amazon RDS**: Managed relational database service that supports MySQL.
- **Amazon S3**: Cloud storage service used for storing static assets like files and images.

## 🔧 Other Tools
- **erdCloud**: Tool for creating and managing ER diagrams.
- **Postman**: API testing tool for verifying request and response workflows.
- **Notion**: Collaboration tool for documenting and planning development workflows.
- **Discord**: Communication tool for team discussions and coordination.
- **Figma**: UI/UX design tool for wireframing and prototyping.
- **GitHub**: Version control and collaboration platform for managing our codebase.
